‚ö° Installation

  1. Unzip the Codecanyon package and copy all files to your web server.
  2. Database:
    • Import: database/real_estate.sql into your MySQL database (use PHPMyAdmin or CLI).
    • Example CLI:
      mysql -u username -p database_name < database/real_estate.sql
  3. Edit DB Config:
    • Edit include/config.php AND admin/include/config.php with your DB credentials:
    $con = mysqli_connect('localhost', 'DB_USERNAME', 'DB_PASS', 'DB_NAME');
  4. Set file permissions (CLI): 
    chmod 755 admin/images/property_image/
Security Tip: After install, login to the admin and change the default password immediately.

‚ùì FAQ / Troubleshooting

  • Can't login to admin? Check your DB, config, and login as above.
  • Images not uploading? Check folder permissions (see above) and PHP file upload limits (in php.ini).
  • Database connection error? Edit config files to match your DB, check error log admin/include/db_error.log.
  • Missing images? System uses placeholder/fallback if images are missing or deleted.
  • Slow site? Use optimal PHP version (8.0+), enable opcache, and use recommended server settings.
